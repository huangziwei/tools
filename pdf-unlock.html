PDF Permissions Unlocker

Remove print, copy, and edit restrictions from PDFs. Runs entirely in your browser — nothing is uploaded.

About & limitations

What this does: Many PDFs are marked “secured” to disallow printing, copying text, or editing, even though the content can be read without a password. The PDF spec still encrypts such files, but with a known-blank user password, so any viewer can read them automatically. This tool uses mupdf.js (a WebAssembly build of MuPDF) to actually decrypt the streams and re-save the PDF without the /Encrypt dictionary or permission flags.

What it cannot do: If a PDF is encrypted with a user password (meaning you need a password to open and read it at all), this tool cannot decrypt it. You would need the password.

Privacy: Everything runs in your browser. Your files never leave your device. The first run downloads ~10 MB of WebAssembly from esm.sh; it’s cached afterward.
