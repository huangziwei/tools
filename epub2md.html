EPUB to Markdown

Convert an EPUB to Pandoc-flavored Markdown in reading order.

About & limitations

What this does: Reads the EPUB's container.xml to find the OPF package, walks the spine in reading order, and emits Pandoc-flavored Markdown.

Preserved: headings, paragraphs, bold, italic, strikethrough (~~x~~), superscript / subscript (^x^ / ~x~), inline code, fenced code blocks, lists (nested), blockquotes, links, tables, horizontal rules, and Pandoc-style footnotes ([^id]) when the source uses EPUB epub:type="noteref" / epub:type="footnote" attributes.

TOC links: cross-chapter links like <a href="chapter01.html"> are rewritten to same-document anchors (#chapter01). An anchor is emitted at the start of every chapter so the links resolve in the merged file.

Images: emitted as ![alt](filename) with the basename only. Image bytes are not included — the output is a single .md file.

Stripped: ruby / furigana (base text kept, reading dropped), underlines, font-size changes, SVG, MathML, arbitrary CSS styling, and any HTML with no Markdown equivalent.

DRM: DRM-protected EPUBs (Adobe ADEPT, Apple FairPlay, Kindle KFX) cannot be read.

Privacy: Everything runs in your browser. Your file never leaves your device.

All data stays on your device.
